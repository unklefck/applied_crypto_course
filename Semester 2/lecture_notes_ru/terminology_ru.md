# Терминология

В книге используется следующая русская терминология.

- **plaintext** — открытый текст
- **ciphertext** — шифртекст
- **encryption** — шифрование
- **decryption** — расшифрование
- **cipher** — шифр
- **block cipher** — блочный шифр
- **stream cipher** — поточный шифр
- **one-time pad** — одноразовый блокнот
- **perfect secrecy** — совершенная секретность
- **information-theoretic security** — теоретико-информационная безопасность
- **computational security** — вычислительная безопасность
- **semantic security** — семантическая безопасность
- **one-way function** — односторонняя функция
- **pseudorandom function** — псевдослучайная функция
- **pseudorandom permutation** — псевдослучайная перестановка
- **Feistel network** — сеть Фейстеля
- **substitution-permutation network** — подстановочно-перестановочная сеть
- **confusion** — перемешивание
- **diffusion** — рассеивание
- **avalanche effect** — лавинный эффект
- **S-box** — S-блок
- **linear cryptanalysis** — линейный криптоанализ
- **differential cryptanalysis** — дифференциальный криптоанализ
- **authenticated encryption** — аутентифицированное шифрование
- **associated data** — ассоциированные данные
- **AEAD** — аутентифицированное шифрование с ассоциированными данными
- **hash function** — хеш-функция
- **preimage resistance** — стойкость к нахождению прообраза
- **second preimage resistance** — стойкость к нахождению второго прообраза
- **collision resistance** — стойкость к коллизиям
- **message authentication code** — код аутентификации сообщения
- **commitment scheme** — схема обязательства
- **zero-knowledge proof** — доказательство с нулевым разглашением
- **secret sharing** — разделение секрета
- **side-channel attack** — атака по стороннему каналу
- **fault attack** — атака с внесением сбоев
- **lightweight cryptography** — легковесная криптография
- **post-quantum cryptography** — постквантовая криптография

Дополнительно используются термины:

- **nonce** — одноразовый номер
- **known-plaintext attack** — атака с известными открытыми текстами
- **chosen-plaintext attack** — атака с выбранными открытыми текстами
- **chosen-ciphertext attack** — атака с выбранными шифртекстами
- **oracle** — оракул

Термины «совершенная секретность» и «семантическая безопасность» не являются синонимами. Первая относится к теоретико-информационной модели и не допускает никакой утечки из шифртекста вообще. Вторая относится к вычислительной модели и означает, что эффективный противник не может извлечь из шифртекста практически полезную информацию с ненезначимым преимуществом.
