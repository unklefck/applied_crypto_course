#### **3. Реализация HMAC**

HMAC должен быть реализован с нуля с использованием собственной реализации SHA-256 из предыдущего спринта.

| ID  | Описание требования                                                                                                 | Приоритет |
| :-- | :------------------------------------------------------------------------------------------------------------------ | :-------- |
| MAC-1 | HMAC **должен** быть реализован студентом с нуля.                                                                  | Must      |
|      | - Реализация **должна** следовать спецификации RFC 2104.                                                            |           |
|      | - В качестве базовой хеш-функции **должна** использоваться реализация SHA-256 из Спринта 4.                       |           |
| MAC-2 | Реализация HMAC **должна** корректно обрабатывать ключ:                                                            | Must      |
|      | - Ключи длиннее размера блока (64 байта для SHA-256) сначала **должны** быть захешированы.                        |           |
|      | - Ключи короче размера блока **должны** быть дополнены нулями.                                                     |           |
| MAC-3 | Реализация **должна** корректно вычислять:                                                                         | Must      |
|      | `HMAC(K, m) = H((K ⊕ opad) ∥ H((K ⊕ ipad) ∥ m))`                                                                   |           |
|      | где `H` — это SHA-256, `opad` — повтор `0x5c`, а `ipad` — повтор `0x36`.                                           |           |
| MAC-4 | Реализация **должна** поддерживать ключи произвольной длины.                                                       | Must      |
| MAC-5 | Реализация **должна** обрабатывать входные файлы по частям для сохранения постоянного потребления памяти.         | Must      |
| MAC-6 | (Бонус) AES-CMAC **может** быть реализован с нуля по NIST SP 800-38B.                                              | Could     |
|      | - Если реализован, он **должен** использовать примитив AES из предыдущих спринтов.                                 |           |
|      | - Он **должен** корректно реализовывать генерацию подключей и конструкцию CBC-MAC с финальным XOR.               |           |

**Ожидаемая структура реализации HMAC:**
```python
# Python: src/mac/hmac.py
class HMAC:
    def __init__(self, key, hash_function='sha256'):
        self.hash_function = get_hash_function(hash_function)
        self.block_size = 64 # bytes, for SHA-256
        self.key = self._process_key(key)

    # Returns SHA256 instance

    def _process_key(self, key):
        # If key is longer than block size, hash it
        if len(key) > self.block_size:
            key = self.hash_function(key).digest()
        # If key is shorter, pad with zeros
        if len(key) < self.block_size:
            key = key + b'\x00' * (self.block_size - len(key))
        return key

    def _xor_bytes(self, a, b):
        return bytes(x ^ y for x, y in zip(a, b))

    def compute(self, message):
        # Create inner and outer pads
        ipad = self._xor_bytes(self.key, b'\x36' * self.block_size)
        opad = self._xor_bytes(self.key, b'\x5c' * self.block_size)
        # Inner hash: H((K ⊕ ipad) ∥ message)
        inner_hash = self.hash_function(ipad + message).digest()
        # Outer hash: H((K ⊕ opad) ∥ inner_hash)
        outer_hash = self.hash_function(opad + inner_hash).digest()
        return outer_hash.hex()
```

**Пример тестового сценария для векторов RFC 4231:**
```python
# tests/test_hmac_vectors.py
from src.mac.hmac import HMAC

def test_rfc_4231():
    """Test HMAC with RFC 4231 test vectors"""
    test_cases = [
        {
            'key': '0b' * 20, # 20 bytes of 0x0b
            'data': '4869205468657265', # "Hi There"
            'expected': 'b0344c61d8db38535ca8afceaf0bf12b881dc200c9833da726e9376c2e32cff7'
        },
        {
            'key': '4a656665', # "Jefe"
            'data': '7768617420646f2079612077616e7420666f72206e6f7468696e673f', # "what do ya want for nothing?"
            'expected': '5bdcc146bf60754e6a042426089575c75a003f089d2739839dec58b964ec3843'
        }
    ]
    for i, test in enumerate(test_cases):
        key = bytes.fromhex(test['key'])
        data = bytes.fromhex(test['data'])
        hmac = HMAC(key, 'sha256')
        result = hmac.compute(data)
        assert result == test['expected'], f"RFC 4231 test case {i+1} failed: got {result}, expected {test['expected']}"
        print(f"RFC 4231 test case {i+1} passed")
```
